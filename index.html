Unity horror prototype

This project is a prototype for a horror game developed in Unity, focusing on immersive item interaction mechanics. I designed and implemented a system that allows players to pick up objects within the game world, examine them closely by rotating them in 3D space, and then return them precisely to their original positions. This feature enhances player engagement and supports puzzle-solving elements, as careful observation and manipulation of items are often crucial in horror games. The system is built with flexibility in mind, making it easy to expand with additional interactive elements as the project evolves.
