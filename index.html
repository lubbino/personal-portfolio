Vibb.online

Vibb.online is a satirical web-based parody of modern dating apps, designed to mock the superficial and repetitive nature of online matchmaking platforms. As the lead developer, I oversaw both the frontend and backend development of the project using PHP, CSS, and JavaScript.

After logging in with demo credentials, users are greeted with humorous profiles, over-the-top bios, and a mock chat system designed to poke fun at real-life dating app clichés. I developed a fully functional login system using PHP sessions, built randomized profile generation logic, and styled the interface to mimic the sleek but sometimes shallow appearance of popular apps like Tinder or Bumble.

I also implemented responsive layouts to ensure mobile usability, and scripted interactive elements to simulate swiping, matching, and messaging features—though all interactions are intentionally exaggerated or broken for comedic effect. The entire experience is self-contained and was designed to highlight both my technical abilities and my creativity when it comes to storytelling and UI design.

This project deepened my understanding of full-stack web development, user session management, and responsive design. It was also an opportunity to experiment with content-driven UX and dark humor in digital interfaces.

Tactile Horror Prototype

This is a prototype for a psychological horror game made in Unity. It focuses on immersive exploration and detailed object interaction, inspired by games like Visage and Layers of Fear.

Item Interaction System

Players can pick up objects, rotate them freely in 3D space, and return them to their original positions. This system encourages close inspection and supports puzzle-solving through environmental storytelling.

Design Purpose

The interaction mechanics let players uncover clues hidden on objects and solve environmental puzzles, reinforcing immersion and tension in the game world.

Technical Approach

Built using modular scripts, a central interaction manager, and temporary physics suspension, the system is designed to be clean, scalable, and flexible for future content.

What I Learned

This project strengthened my skills in Unity input systems, 3D object manipulation, and designing immersive mechanics that contribute to gameplay and narrative.

RPi remote control with camera

This project is a web-controlled robot built using a Raspberry Pi, created in collaboration with Albin Persson. It features both a Python-based client for hardware control and a PHP-based server for the browser interface.

Users can connect through a web browser to remotely view the robot’s camera feed and send movement commands. The client script interprets these commands to control motors and manage real-time video streaming from the Raspberry Pi’s camera module.

The project demonstrates a practical implementation of robotics, networking, and browser-based control, making it a hands-on exploration of IoT systems using accessible technologies.
